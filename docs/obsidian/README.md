# Obsidian vault

This folder is an Obsidian-ready knowledge base for the project. Notes use
wikilinks and YAML frontmatter, so they render best inside Obsidian but stay
readable on GitHub.

## Open it in Obsidian

Either:

1. **Standalone vault** — Obsidian → *Open folder as vault* → select
   `docs/obsidian/`.
2. **Inside an existing vault** — symlink it in:

   ```bash
   ln -s /path/to/sa-fintech-skills/docs/obsidian ~/ObsidianVault/sa-fintech-skills
   ```

Start at [[Home]].

Per-machine Obsidian workspace state (`.obsidian/`) is gitignored on purpose —
themes and pane layouts are yours, the notes are shared.
